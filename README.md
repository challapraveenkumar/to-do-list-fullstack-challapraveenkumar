✅ MERN To-Do List Application

This is a full-stack To-Do List application built using the MERN Stack (MongoDB, Express.js, React.js, Node.js). The application allows users to create, manage, update, and delete tasks efficiently. It provides a responsive user interface and a RESTful backend API for task management.

🚀 Features

➕ Add new tasks

✏️ Update existing tasks

❌ Delete tasks

📋 View all tasks

💾 Store tasks in MongoDB database

⚡ Fast and responsive React frontend

🔗 RESTful API using Node.js and Express
